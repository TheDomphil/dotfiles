----------------------------------------------------------------------------

MAN PAGES

Man pages are incredibly useful pages

Man pages are opened by typing:

man <name_of_something>

If you want to open the fifth man page of something:

man 5 <name_of_something>

----------------------------------------------------------------------------
GIT

To even run git first you need to:

eval $(ssh-agent)

then you can add your key:

ssh-add <path/to/ssh/key> (typically in ~/.ssh/<keyname>)

then:

git add .
git commit -m "Message"
git push
